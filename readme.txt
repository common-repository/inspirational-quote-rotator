== Description ==

If you're looking for the perfect way to inspire your visitors, the Inspirational Quote Rotator is the perfect plugin for you.  
Just activate and each visitor will see an inspirational quote appear when he visits your website.  
This plugin is ideal for any website that seeks to inspire the best in its visitors, like:  
*   Motivational speakers  
*   Sports blogs  
*   Business blogs  
*   Self-help sites  
*   Or in our case, life coaching blogs  


== Installation ==

1. Unzip archive with plugin and upload it to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
